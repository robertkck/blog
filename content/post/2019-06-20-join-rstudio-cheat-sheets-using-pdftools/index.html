At the occastion of the RISIS Summer School on Data Science in Science, Technology and Innovation Studies I wanted to print all RStudio cheatsheets. They are very well designed, speed up the coding and provide a good overview of the main functionality of a package. The regex, ggplot2, purrr and RMarkdown cheatsheets are my go to source.

Garrett Grolemund at RStudio maintains the github repo of all contributed cheatsheets. Since the sheets have been contributed by members of the R Community under CC-BY-4.0, I can share my merged files with you. You will find credits for each individual sheet author at the bottom of the page.

Either download each cheatsheet into a folder or clone the github repo mentioned above (>900MB). Then, using the pdftools package by Jeroen Ooms, I merged the one-sided and two-sided cheatsheets into to individual PDFs for easy printing :printer:

f <- list.files("cheatsheets/", pattern = "\\.pdf$", full.names = TRUE)
len <- purrr::map_int(f, pdftools::pdf_length)
pdftools::pdf_combine(f[len == 2], output = "2sided.pdf")
pdftools::pdf_combine(f[len == 1], output = "1sided.pdf")
